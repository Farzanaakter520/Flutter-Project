# Hospital_APP (Flutter)

## Project Overview
The **Hospital Management System** is a mobile application developed using **Flutter** to digitally manage patient, doctor, and staff information for hospitals and clinics. It streamlines hospital operations, including patient admission, appointments, doctor schedules, and medical records. Being Flutter-based, it works seamlessly on both Android and iOS platforms.
